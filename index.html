Welcome to PyHachi’s documentation!¶

Introduction¶

PyHachi: A simple but scalable content filter framework

Todo

Features¶

Note

  • Char level filter (will update to Wu-Manber soon, ref: https://guokr.github.io/min-gan-ci-xiang-mu-zhong-rou-xing-zi-fu-chuan-pi-pei-wen-ti.html)
  • Word level filter powered by THULAC-lite (ref: https://github.com/thunlp/THULAC-Python)
  • Pattern matching based on regex(ref: https://github.com/kemingy/cnprep)

Requirements and Installation¶

pip install pyhachi

Getting Started¶

#!/usr/bin/env python3
# encoding: utf-8

from pprint import pprint
from pyhachi import HachiModel
from pyhachi import BlacklistPlugin, PatternPlugin

plug_1 = BlacklistPlugin(words_list=["可乐", "雪碧", "星空闪亮草莓奶昔", "av", "java"])
print(plug_1)

# <PLUGIN>: blacklist_982dek,
# <TYPE>: BlacklistPlugin,
# <WORD COUNT>: 5

plug_2 = PatternPlugin()
print(plug_2)

# <PLUGIN>: pattern_max0zn,
# <TYPE>: PatternPlugin,
# <PATTERN>: email,telephone,url,QQ,wechat

model = HachiModel(plugins=[plug_1, plug_2])

query = ("我这里有雪碧，可乐饼，星空闪亮草莓奶昔和许多其他的，你要不要尝尝？\
         我的手机号是13100000000 我的个人网站www.guokr.com\
         123o4usdjfk@gmail.com, or sfjjsaf@163.com, java")

res = model.check(query)

pprint(res)
